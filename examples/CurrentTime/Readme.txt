This file is part of Almagesto, a Free Pascal astronomical library. 

This example gets the current UTC date and time and converts it to the time scales implemented in the class TTimeScales from the unit almDateTime.

Required packages: LCL and almagesto.

Tested with the following Lazarus and FPC versions:
- Lazarus 1.0.4 FPC 2.6.0 x86_64-linux-gtk2
